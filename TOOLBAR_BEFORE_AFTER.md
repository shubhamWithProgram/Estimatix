### **4. Interaction Animations**

#### BEFORE
```css
/* Minimal or no animations */
button {
  transition: background-color 0.15s;
}
```

#### AFTER
```css
.action-btn {
  transition: all 0.3s cubic-bezier(0.4, 0, 0.2, 1);
}

/* Hover: Lift + Scale + Glow */
.action-btn:hover:not(:disabled) {
  transform: translateY(-2px) scale(1.02);
  box-shadow: 0 6px 16px rgba(0, 0, 0, 0.15);
}

/* Active: Press down */
.action-btn:active:not(:disabled) {
  transform: translateY(0) scale(0.98);
}

/* Special hover for AI button */
.btn-warning-subtle.action-btn:hover:not(:disabled) {
  transform: translateY(-2px) scale(1.03); /* Slightly larger */
  box-shadow: 0 6px 20px rgba(255, 193, 7, 0.35);
}
```

**Effect:**
- âœ… **Tactile feedback** - Button feels clickable
- âœ… **Smooth transitions** - 0.3s cubic-bezier
- âœ… **Depth on hover** - Lift animation
- âœ… **Press sensation** - Scale down on click
