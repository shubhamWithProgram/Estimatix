## 💰 Quick Calculations

### Area:
```
Area (m²) = Width × Height × Quantity
```

### Frame Weight:
```
Standard:      2.5 kg/m
Heavy Duty:    3.5 kg/m
Powder Coated: 2.8 kg/m
Wooden:        2.0 kg/m
UPVC:          1.8 kg/m

Weight = Perimeter × Density × Qty
```
